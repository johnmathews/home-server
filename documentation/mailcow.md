Mailcow is a Docker-based mail server suite running on VM 103.

- **Domain:** itsa.pizza
- **Mail hostname (FQDN):** mail.itsa.pizza
- **VM ID:** 103
- **IP:** 192.168.2.103
- **OS hostname:** mailcow
- **DNS provider:** Cloudflare (tunneled via cloudflared LXC)
- **Install path:** /opt/mailcow-dockerized

## Access

SSH into the VM:

```sh
ssh root@192.168.2.103
```

Mailcow web UI: https://mail.itsa.pizza

## DNS Records

Required DNS records in Cloudflare for itsa.pizza:

```
Type   Name              Value
A      mail.itsa.pizza   <public IP>
MX     itsa.pizza        mail.itsa.pizza (priority 10)
TXT    itsa.pizza        v=spf1 mx -all
TXT    dkim._domainkey.itsa.pizza   <generated by mailcow>
TXT    _dmarc.itsa.pizza            v=DMARC1; p=reject; rua=mailto:postmaster@itsa.pizza
```

The DKIM key is generated after first run. Retrieve it from the mailcow UI under System > Configuration > Options > ARC/DKIM keys.

## Managing the Stack

```sh
cd /opt/mailcow-dockerized
docker compose up -d
docker compose down
docker compose logs -f
```

## Updating Mailcow

```sh
cd /opt/mailcow-dockerized
./update.sh
```

## Configuration

Main config file generated by `./generate_config.sh`:

```sh
/opt/mailcow-dockerized/mailcow.conf
```
